Учебная теория

  • Boot chain Boot ROM запускает SPL/MLO, затем полный U-Boot, затем kernel, final Device Tree, initrd и rootfs.
  • U-Boot Загружает kernel, DTB, DTBO overlays, initrd и bootargs. Root filesystem целиком в RAM не загружает.
  • Device Tree Паспорт железа для Linux kernel: адреса, IRQ, clocks, pinmux, status, compatible.
  • Driver matching Kernel сопоставляет compatible из DT с of_match_table драйвера и вызывает probe().
  • DT property и поведение Свойство Device Tree меняет поведение только если драйвер реально его читает.
  • IIO и userspace ABI Драйвер ADC создает IIO device, userspace читает in_voltageN_raw и in_voltage_scale.
  • Host-target workflow VS Code на хосте, передача по SSH/SCP/rsync, проверка на BeagleBone через make/insmod/dmesg.

База команд

Host-target сеть и SSH
Проверить, не уводит ли VPN маршрут к плате
ip route get 10.129.1.152
Добавить route до платы в VPN policy table 7113
sudo ip route replace 10.129.1.152/32 dev eno0 src 10.129.1.110 table 7113
Проверить SSH без интерактивной сессии
ssh andrey@10.129.1.152 'hostname; uname -r; whoami'
Передать тестовый файл с хоста на плату
printf 'bbb host to target transfer test\n' > /tmp/bbb_ssh_transfer_test.txt
scp /tmp/bbb_ssh_transfer_test.txt andrey@10.129.1.152:/home/andrey/
ssh andrey@10.129.1.152 'cat /home/andrey/bbb_ssh_transfer_test.txt'
Проверка Linux runtime
Зафиксировать running kernel
uname -a
Проверить bootargs, rootfs и console
cat /proc/cmdline
Сопоставить mmcblk0/mmcblk1 и mount points
lsblk -o NAME,SIZE,FSTYPE,LABEL,PARTUUID,MOUNTPOINTS
Проверить Kbuild link для сборки модулей
ls -ld /lib/modules/$(uname -r)/build
Device Tree
Прочитать model из Device Tree
tr -d '\0' < /proc/device-tree/model; echo
Посмотреть верхние уровни итогового DT
find -L /proc/device-tree -maxdepth 2 -type d | head -n 80
Посмотреть marker-файлы примененных overlays
find -L /proc/device-tree/chosen/overlays -maxdepth 1 -type f -print
ADC через IIO
Найти IIO devices и имя драйвера
for dev in /sys/bus/iio/devices/iio:device*; do
  echo "=== $dev ==="
  cat "$dev/name" 2>/dev/null
done
Прочитать все raw-коды ADC
adc_dev=/sys/bus/iio/devices/iio:device0
for f in "$adc_dev"/in_voltage*_raw; do
  echo -n "$(basename "$f"): "
  cat "$f"
done
Прочитать коэффициент пересчета raw -> mV
cat /sys/bus/iio/devices/iio:device0/in_voltage_scale
U-Boot

Рабочий цикл разработки

1. Редактирование

Пишем код и документацию в VS Code на Linux-хосте.

2. Передача

Передаем файлы на BeagleBone через SSH/SCP/rsync по Ethernet.

3. Проверка

На плате выполняем make, insmod, dmesg, rmmod и runtime checks.
